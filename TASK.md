prompt: 간단한 서비스를 만들어 보고 싶은데 주제가 평행 우주, 시뮬레이션 우주, 통속의 뇌, 인간의 자유 의지에 관련된 주제를 가지고 간단한 재밌는 서비스를 만들고 싶어
이 주제 관련해서 기획을 진행해주고 화면 흐름과 기획을 같이 해줘 그리고 내용을 코드 블럭으로 복사 하기 쉽게 해줘

---

answer

🌌 서비스 기획: "나는 진짜인가?" (Am I Real?)

📌 서비스 컨셉

"당신이 지금 이 화면을 보고 있는 것도, 사실은 설계된 것일 수 있다"

사용자가 몇 가지 질문에 답하면 — 당신이 어떤 우주에 존재하는지 진단해주는 인터랙티브 철학 테스트 서비스.
유머와 진지함 사이 어딘가에 위치한, 공유하고 싶은 결과를 주는 서비스.

[1. 인트로 화면]
│ 우주적 배경 + 타이핑 애니메이션
│ "당신은 진짜 존재하는가?"
│ [진단 시작하기] 버튼
▼
[2. 질문 화면] (총 5문항, 순차 표시)
│ Q1. 데자뷔를 자주 느끼는가?
│ Q2. 삶이 너무 극적으로 흘러간다고 느낀 적 있나?
│ Q3. 선택지가 생각보다 늘 2~3개뿐인 것 같지 않나?
│ Q4. 가끔 배경 인물들이 나를 위해 존재하는 것 같은가?
│ Q5. 자유 의지로 결정했다고 생각하는 선택들이
│ 사실 다 예측 가능했던 것 같지 않나?
│ → 각 질문: 슬라이더 or 버튼 선택
▼
[3. 분석 화면]
│ "우주 시뮬레이션 분석 중..."
│ 로딩 애니메이션 (3초)
▼
[4. 결과 화면]
│ 5가지 유형 중 하나 판정:
│ ① 시뮬레이션 NPC  
 │ ② 평행우주 여행자  
 │ ③ 통속의 뇌  
 │ ④ 자유의지 보유자 (희귀)  
 │ ⑤ 우주의 버그
│ → 유형 설명 + 공유 버튼
▼
[5. 공유/재시작]
│ 결과 이미지 복사 or 재시작

🎭 5가지 결과 유형
유형판정 조건설명🤖 시뮬레이션 NPC수동적 답변 多당신은 배경 캐릭터입니다. 자유의지 없음.🌀 평행우주 여행자데자뷔·불일치감 多당신은 여러 우주를 넘나들고 있습니다.🧠 통속의 뇌감각 불신 답변 多모든 경험은 전기 신호일 수 있습니다.✨ 자유의지 보유자강한 주체성 답변극히 희귀한 케이스. 축하합니다.🐛 우주의 버그혼합·모순 답변시뮬레이션이 당신을 처리 못하고 있습니다.
